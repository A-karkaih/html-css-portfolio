# Personal Portfolio (HTML & CSS)

This is a portfolio website built using **pure HTML and CSS**.
The goal of this project was to practice and demonstrate fundamental **front-end development skills** without using any frameworks or JavaScript.

## 🚀 Features

* Responsive layout
* Modern UI design
* Clean and structured code

## 🛠️ Technologies Used

### HTML

* Semantic HTML structure
* Forms
* Images
* Sections and layout elements

### CSS

* Flexbox layout
* Custom fonts
* Icons
* Styling forms
* Responsive design
* Image styling
* Hover effects

## 📚 What I Practiced

* Building a complete website using only **HTML and CSS**
* Structuring pages with **semantic HTML**
* Creating responsive layouts using **Flexbox**
* Styling **forms, images, and icons**
* Working with **custom fonts**
* Organizing CSS for maintainability

## 📷 Preview

Portfolio sections include:

* NavBar
* Hero Section
* Skills
* Testimony
* Contact form

## 🎯 Purpose

This project was created to strengthen my **front-end fundamentals** and showcase my ability to build a clean and responsive interface using core web technologies.
